# Healthcare Appointment Overbooking Stabilizer
## Problem Statement
Hospitals often overbook appointments to improve efficiency, which leads to long waiting times, scheduling conflicts, and patient dissatisfaction. There is a need for a system that can manage overbooked appointments effectively and ensure fair and smooth scheduling.

## Proposed Solution
This project provides a smart healthcare appointment system that manages overbooking using intelligent scheduling. It helps in predicting wait times, handling cancellations, and automatically rescheduling patients while maintaining fairness and minimizing disruption.

## Technology Stack
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

## Setup Instructions
1. Install XAMPP  
2. Place project folder inside htdocs  
3. Start Apache and MySQL  
4. Import database into phpMyAdmin  
5. Run http://localhost/healthcare
